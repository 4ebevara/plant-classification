ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ К КОДУ:
Создай два файла: helpers.py (вспомогательные функции) и generate_diploma.py (контент)
helpers.py должен содержать функции:
init_document() — создание документа с настройками
set_page_margins(section) — поля страницы
set_run_font(run, size, bold, italic, font_name) — форматирование текста с принудительным rFonts
set_paragraph_format(p, ...) — форматирование абзаца
add_body_text(doc, text) — абзац основного текста
add_heading_section(doc, text) — заголовок раздела (с новой страницы, ПРОПИСНЫЕ)
add_heading_sub(doc, text) — заголовок подраздела
add_table_caption(doc, number, title) — подпись таблицы
add_figure(doc, number, caption, image_path) — вставка рисунка с подписью
make_table(doc, headers, rows) — создание таблицы
add_dash_item(doc, text) — пункт списка с тире
add_page_numbers(doc) — нумерация (пропуск 1-й секции, нумерация со 2-й)
В generate_diploma.py:
Отдельная функция для каждого раздела: write_contents(), write_introduction(), write_section_1_1() и т.д.
main() собирает документ: init → contents → add_section → introduction → chapters → conclusion → references → page_numbers → save
Текст должен быть содержательным и по теме — не менее 4–6 абзацев на подраздел
Код должен быть сразу запускаемым: python generate_diploma.py → готовый .docx
Сгенерируй полный код обоих файлов. Текст ВКР должен быть осмысленным, академическим, без воды.
