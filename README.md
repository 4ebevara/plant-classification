# Классификация растений по изображениям

**ВКР: Классификация растений по изображениям с использованием алгоритмов компьютерного зрения для агротехнического контроля**

**Автор:** Иманалиев Асылбек

---

## Описание

Система автоматической классификации заболеваний растений на основе фотографий листьев.  
Используется модель **EfficientNet-B0** с технологией Transfer Learning, обученная на датасете **PlantVillage** (38 классов, ~54 000 изображений).

### Возможности
- Классификация 38 типов заболеваний 14 сельскохозяйственных культур
- Веб-интерфейс для интерактивной диагностики
- Визуализация областей внимания модели (Grad-CAM)
- Рекомендации по лечению

## Быстрый старт (на любом компьютере)

### 1. Клонирование репозитория

```bash
git clone https://github.com/YOUR_USERNAME/plant-classification.git
cd plant-classification
```

### 2. Создание виртуального окружения

```bash
python -m venv venv

# Windows:
venv\Scripts\activate

# Linux/Mac:
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

> **Примечание:** PyTorch для CPU устанавливается автоматически. Для GPU-версии см. [pytorch.org](https://pytorch.org/get-started/locally/)

### 4. Запуск веб-приложения (демо)

```bash
streamlit run app.py
```

Откроется браузер с интерфейсом. Загрузите фото листа — система покажет диагноз, уверенность и Grad-CAM карту.

### 5. Запуск Jupyter Notebook

```bash
jupyter notebook plant_classification.ipynb
```

### 6. Обучение модели с нуля

```bash
python run_training.py
```

> Датасет PlantVillage скачивается автоматически через `kagglehub`.
